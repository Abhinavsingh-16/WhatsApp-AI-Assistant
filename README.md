Built an end-to-end WhatsApp AI automation workflow integrating Meta's WhatsApp Business API, OpenAI (GPT-4.1-mini), and Google Calendar. The system receives WhatsApp messages via webhook, processes them through an AI model, and responds intelligently — including booking and managing Google Calendar events directly from WhatsApp chat.
Key highlights:

• Facebook Developer App configured for WhatsApp Business API

• GPT-4.1-mini used for natural language understanding and response

• Google Calendar integrated for real-time scheduling via chat

• Full webhook handling with n8n automation engine
This project reflects product thinking around conversational AI interfaces and real-world API integration.
